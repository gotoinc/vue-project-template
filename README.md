# Vue 3 + TypeScript + Vite

## Recommended setup
- [Volta.sh](https://volta.sh/)

## Project set up

```bash
# Install dependencies
$ yarn install 

# Compiles and hot-reloads for development
$ yarn dev

# Compiles and minifies for production
$ yarn build
```

## Prepare local environment

```bash
$ yarn setup-husky
```

```bash
$ git config core.hooksPath .husky
```
